# 图片颜色抓取器
## 使用帮助
* 双击打开根目录下的index.html（需要支持html5的浏览器）
* 将图片拖入到暂无图片的位置，此时工具会加载并显示图片
* 在图片区域移动鼠标，会显示放大后的图片局部
* 单击鼠标左键，会将当前的坐标颜色，记录到特征列表
* 拖动鼠标左键，会拖动图像
* 按住ctrl+单击鼠标左键，会放大放大镜范围
* 按住ctrl+拖动鼠标左键，会选取范围，选取的范围会自动复制到剪切板
* 按住ctrl+单击鼠标右键，会缩小放大镜范围
* 单击鼠标右键会取消选取的范围
* 按上下左右键，会使当前指向位置移动一个像素

* 在特征框中间位置单击鼠标左键，会复制当前的坐标与颜色
* 在特征框偏左位置单击鼠标左键，会复制当前的坐标
* 在特征框偏右位置单击鼠标左键，会复制单前的颜色
## 开发帮助
### 前提条件
* 需要电脑安装nodejs
* 需要电脑安装npm
### 自定义生成格式
参照[main.ts](./src/main.ts)文件中的后几行便可以知道如何生成和设置剪切板等，
修改后，在项目目录下运行命令行 `npm run build`,重新生成[main.js](./dist/main.js)
双击index.html，此时修改已经生效
